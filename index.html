JobFolder | Full-Stack

I love to make tools! I wanted to make a tool that simplified the lives of job-seekers. When a person is in full-time job-search mode, it is recommended that he apply to fifteen jobs a day! That kind of volume demands being organized. So this full-stack app brings React, Node, Express, and Postgres together to provide one smart place for all the job-seeker's documents, interview questions, and contacts. The really cool thing about this app is what it does with the info the user gives it. It sends interview pop-quizzes and emails to remind the user to follow up and send thank-you notes.

DC Quiz App

Another helpful tool I made, a robust quiz app. It's a critical discipline when learning a new subject is to regularly quiz oneself on the content. This server-side rendered full-stack app brings together HTML, CSS, JavaScript, Node, Express, and ES6 Template Engine allows users to take a variety of quizzes on different development-related topics. The users can select length of the quiz and the topic. The quiz doesn't merely give a score at the end but gives valuable feedback about what was missed. There is also a leaderboard that keeps track of scores over time comparing them with other users. Users can also submit questions to the database.

Express App Generator | Shell

I love making automation tools. This Express App Generator scaffolds an Express app in seconds allow the developer to get right down to building. It creates the following folders: Project Directory named based on user input, Views, Models, Public, Routes, Config. It creates an app.js fully stocked with everything needed for basic server function: http, express, morgan, nodemon. It creates, .env, style.css, and .sequelizerc. It sets up and configures express-es6-template-engine. It initializes the project as a node package with 'npm init -y.' It protects node-modules, and .env in .gitignore. It sets up one basic route telling you that you are ready to build.
